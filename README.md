# Love Booth

Love Booth is a modern, web-based photo booth application that allows users to create stunning photo strips effortlessly. Whether you're at a party, a wedding, or just hanging out with friends, Love Booth lets you capture moments or upload photos to generate professional-looking photo strips instantly.

![Love Booth Logo](./src/assets/lovebooth-logo.png)

## üåü Features

- **Flexible Layouts:** Choose from multiple photo strip layouts:
  - **Classic Strip:** Vertical strip with 3 or 4 photos.
  - **Grid:** 2x2 grid layout.
  - **Wide:** Cinema-style wide shots.
- **Capture Options:**
  - **Live Camera:** Take photos directly within the app using your device's camera.
  - **Countdown Timer:** Built-in 3s, 5s, and 10s timer for the perfect pose.
  - **Upload:** Select existing photos from your gallery.
- **Customization:**
  - **Filters:** Switch between **Color** and **Black & White** modes instantly.
  - **Preview:** Real-time preview of your photo strip before saving.
- **high-Quality Output:**
  - **Download:** Save high-resolution, print-ready PNG files.
  - **Bluetooth Printing:** Connect directly to Bluetooth thermal printers for instant physical copies.
  - **Print Support:** Standard browser print integration.

## üöÄ Getting Started

Follow these steps to set up the project locally on your machine.

### Prerequisites

- [Node.js](https://nodejs.org/) (Version 18 or higher recommended)
- [npm](https://www.npmjs.com/) (usually comes with Node.js)

### Installation

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/Tusharjain-19/Love-Booth.git
    cd Love-Booth
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    ```

3.  **Start the development server:**

    ```bash
    npm run dev
    ```

4.  **Open your browser:**
    Navigate to `http://localhost:8080/` to see the app in action.

## üì± Usage

1.  **Select a Layout:** Choose your preferred photo arrangement (Strip, Grid, or Wide).
2.  **Add Photos:**
    - Click "Use Camera" to snap new pictures.
    - Click "Upload" to use images from your device.
3.  **Capture:** If using the camera, get ready! The timer will count down for each shot.
4.  **Edit & Save:**
    - In the preview screen, toggle between Color and B&W.
    - Click **Download** to save the image to your device.
    - Click **Print** to send to a connected printer.
